# Summarize

Use the LLM to summarize long text.

## Usage

Send text to the agent with instruction to "summarize this" or "give me the key points".

The agent will use its language model to extract key information and present a concise summary.

## Tips

- Provide the full text you want summarized
- Specify focus areas: "summarize the main arguments" or "extract action items"
- Ask for bullet points if preferred
